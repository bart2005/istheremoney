Когда пытался добавить яндекс тэг для проверки принадлежности сайта (вот здесь), я наткнулся (я и раньше знал, просто забыл напрочь) на валидатор html validator.w3.org. Красивое, но это онлайн, руками надо лазить, что за дела. Сайт подсказывает, что он работает на основе Nu Html Checker (vnu), который всем доступен и бесплатен ( буржуи, конечно, денег валом:)) Скачиваю себе java версию (смотри здесь) и добавляю к своей мощной автоматизированной системе деплоя, которая пока состоит из одного скрипта (UPD: перевел на make, как-то поудобнее).

Мой текущий процесс деплоя

make build verify - проверяю валидность
make update-time - обновляют время и генерю sitemap
gcam "sofisticated git message" - коммит
gp - пуш
